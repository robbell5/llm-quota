<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| local files → refresh command | Claude cache and Codex rollout files are untrusted local inputs parsed by source readers before entering TUI state. |
| refresh goroutines → Bubble Tea Update | Concurrent file reads return messages; only `Update` mutates model state. |
| terminal input → Update | Keypress and resize messages are untrusted terminal events. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | `refreshCmd` source results | mitigate | Keep source parsing in `internal/sources`; merge only normalized `[]sources.Window` and typed errors. |
| T-03-02 | Denial of Service | duplicate manual refresh | mitigate | Use `refreshing` state so `r` while a refresh is running returns nil command. |
| T-03-03 | Information Disclosure | tests and refresh defaults | mitigate | Tests use fake readers and never read home-directory paths or real quota files. |
| T-03-04 | Repudiation | source failure state | mitigate | Preserve per-source typed errors in model state for later rendering and tests. |
</threat_model>

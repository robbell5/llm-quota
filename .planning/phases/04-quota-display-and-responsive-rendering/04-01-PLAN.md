## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| local source data → TUI renderer | Parsed quota percentages and reset timestamps are local but still untrusted for display width and range safety. |
| terminal renderer → user | ANSI-styled output must stay bounded to the pane width and must not obscure the true quota value. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-04-01-02 | Denial of Service | `internal/tui/view.go` width calculations | mitigate | Compute bar width from available row width and verify ANSI-stripped line widths at width 50 and wider. |
